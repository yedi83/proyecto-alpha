# research/

Laboratorio: todo el trabajo práctico de cada hipótesis, organizada por carpeta `H-XXX-nombre/`.

Reglas (ver `docs/PLAN_TRABAJO.md` §2):
1. Las hipótesis **nunca cambian de carpeta** por su veredicto; el estado vive en el registro y la ficha.
2. Experimentos append-only y numerados. Nada se borra: las rechazadas son biblioteca, no basura.
3. Código validado se **promueve** (copia congelada) a `strategies/`; esta carpeta sigue siendo el laboratorio.
4. Nueva hipótesis = copiar `_template/` + ficha en docs + fila en el registro.
